# buengx.github.io

Personal website and portfolio showcasing my work and projects.

## About

This is my personal website hosted on GitHub Pages. It serves as a place to introduce myself, showcase my skills, and share my projects.

## Features

- **Personal Introduction**: Learn about who I am and what I do
- **Skills & Technologies**: Overview of my technical expertise
- **Projects**: Dynamic display of my GitHub repositories
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean, professional design with smooth animations

## Site Structure

- `index.html` - Main homepage with personal introduction
- `styles.css` - CSS styling and responsive design
- `script.js` - JavaScript functionality and GitHub API integration
- `404.html` - Custom 404 error page
- `README.md` - This documentation

## Local Development

To view the site locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. Or serve it with a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

## Live Site

This site is automatically deployed to GitHub Pages at: `https://buengx.github.io` (or `https://buengx.tech`)

## Technologies Used

- HTML5
- CSS3 (with CSS Grid, Flexbox, and modern styling)
- Vanilla JavaScript
- GitHub Pages for hosting
- GitHub API for dynamic repository display
